🚀 Steve Bot V1.0

AI-Powered Trading Coordinator — Setup Guide
⚡ 10-15 min setup
1
Create your Oracle Cloud account
~5 min

Oracle Cloud gives you a free virtual machine that runs Steve Bot 24/7. No credit card charges — you're on the free tier.

  1. Open cloud.oracle.com on your phone
  2. Tap "Sign Up for Free Tier"
  3. Enter your email, create a password
  4. They'll ask for a card to verify your identity — a small hold is placed and refunded. You won't be charged.
  5. Enter your phone number for SMS verification
📎 Keep this open

Leave Oracle Cloud logged in. The next step uses it.

2
Create your virtual machine
~5 min

This is the computer that Steve Bot lives on. Think of it as a rental server in Oracle's data centre.

  1. Tap the menu (☰) → "Compute" → "Instances"
  2. Tap "Create Instance"
  3. Name: stevebot
  4. Scroll to "Image and Shape" → tap "Change Shape"
  5. Select the "Ampere A1" tab → VM.Standard.A1.Flex with 4 OCPU, 24 GB RAM
  6. Under "Add SSH Keys" → tap "Generate SSH Key Pair"
  7. Tap the "Save Private Key" and "Save Public Key" buttons
  8. Tap "Create"
  9. Wait ~2 minutes while your VM starts up
⚠️ Critical — Save Your Keys

The SSH private key is your only way to access the VM. If you lose it, you'll need to delete the VM and start over. Save it to your phone's Downloads.

3
Install an SSH app on your phone
~1 min
iPhoneTermius (App Store, free)
AndroidTermius or JuiceSSH (Play Store, free)

Install the app, then come back here.

4
Connect to your VM
~2 min

Open Termius (or your SSH app). You need two things from Oracle Cloud:

Host / IP(from Step 2)
Usernameubuntu
Auth methodSSH Key (import saved file)
  1. Open Termius → "New Host"
  2. Find your VM's Public IP Address under "Instance Details" in Oracle Cloud — it looks like 123.123.123.123
  3. Enter that IP in the Host field
  4. Username: ubuntu
  5. Under "SSH Key" → "Import Key" → select the private key file you downloaded from Oracle
  6. Tap Connect
✅ You should see

A command prompt like ubuntu@stevebot:~$ — this means you're connected!

🔍 Can't find your IP?

In Oracle Cloud, go to Compute → Instances → tap your stevebot instance. The Public IP is at the top of the page.

🔑 Before We Run the Installer

You need 3 API keys. Get them now — the wizard will ask for them in the next step.

🔑
DeepSeek API Key
~2 min

This is how you pay for Steve Bot's AI brain. Costs about $3-5/month.

  1. Open platform.deepseek.com
  2. Sign up (email + password)
  3. Tap "Top Up" → add $10 (lasts 2-3 months)
  4. Go to "API Keys" → "Create new key"
  5. Copy the key — it starts with sk-
📌 Keep this tab open

You'll paste this key into the setup wizard in the next step.

🔑
Bitget API Key
~3 min

Connects Steve Bot to your exchange account. Your funds stay safe.

  1. Log into bitget.com on your phone
  2. Tap your profile icon → "API Keys" → "Create API Key"
  3. Name: stevebot
  4. Permissions: Enable only "Futures Trading"
  5. ❌ DO NOT enable "Withdraw"
  6. Save the API Key, Secret Key, and Passphrase
⚠️ Security

With no withdrawal permission, Steve Bot can trade but can never take your funds. Even if someone hacked the VM, your money stays on Bitget.

🔑
Telegram Bot Token
~2 min

This is how you talk to Steve Bot. You'll create a bot in Telegram and get a token.

  1. Open Telegram, search for @BotFather
  2. Send /newbot
  3. Name: Steve Bot
  4. Username: pick something unique like YourNameBot
  5. BotFather replies with a token like 123456:ABCdef...
  6. Copy the token

Also get your Chat ID:

  1. Search for @userinfobot
  2. Send /start
  3. It replies with your Chat ID — a number like 123456789
  4. Copy the Chat ID
6
Install Steve Bot
~3 min

Back in Termius (your VM), copy-paste this one command and press Enter:

curl -sSL https://stevebot-updates.pages.dev/install.sh | bash

The installer will:

  • Install Python, Node.js, and OpenClaw
  • Create the Steve Bot directory structure
  • Launch the interactive setup wizard
📌 How to paste in Termius

Tap once in the terminal → tap "Paste". Or long-press → Paste.

7
The Setup Wizard
~2 min

The wizard asks for 7 things. Just copy-paste each API key when asked. The wizard tests each one live.

1. Your Name"Ross"
2. DeepSeek API Keysk-... (pastes hidden)
3. Bitget API Keybg_...
4. Bitget Secret(pastes hidden)
5. Bitget Passphrase(pastes hidden)
6. Telegram Bot Token123456:ABC...
7. Your Chat ID123456789

Each key is tested immediately. If something's wrong, the wizard tells you and lets you retry.

✅ After the wizard finishes

Steve Bot starts automatically. Open Telegram and send /start

✅ You're Done!

Steve Bot is running. Open Telegram and say hello.

He'll respond with "🦈 Morning, Champ!"
